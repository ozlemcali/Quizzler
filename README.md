# Quizzler
a simple true-false game


## About Project
I made a quiz app. Quizzler is a true false quiz app about cities and regions.
## Includes

* Object Oriented Programming
* Struct-Class
* UIAlertControllers
* MVC Design Pattern
* UI elements such as Labels and Views
* Colors
